## Color Combinations

### High Contrast (Accessible)
```
Text: #10100F (Black) on #FFFFFF (White)          ✓ WCAG AAA
Text: #FFFFFF (White) on #5A7554 (Envy 500)       ✓ WCAG AA
Text: #FFFFFF (White) on #598BD8 (Spindle 500)    ✓ WCAG AA
Text: #10100F (Black) on #F4F6F3 (Envy 50)        ✓ WCAG AAA
```

### Medium Contrast
```
Text: #BE8A75 (Wafer 500) on #FFFFFF (White)      ✓ WCAG AA
Text: #5A7554 (Envy 500) on #F4F6F3 (Envy 50)     ✓ WCAG AA
```

### Complementary Pairs
```
Primary + Secondary:  Envy 500 (#5A7554) + Wafer 500 (#BE8A75)
Brand + Accent:       Spindle 500 (#598BD8) + Envy 500 (#5A7554)
Light + Dark:         Envy 50 (#F4F6F3) + Envy 950 (#141B13)
```

## Usage Guidelines

### Backgrounds
- **Main**: White (#FFFFFF)
- **Alternate**: Envy 50 (#F4F6F3)
- **Cards**: White with shadow or Envy 100 (#E5EAE1)
- **Panels**: Envy 50 to Envy 200

### Text
- **Primary**: Black (#10100F)
- **Secondary**: Wafer 500 (#BE8A75)
- **Disabled**: 50% opacity of primary
- **On Dark**: White (#FFFFFF)

### Buttons
- **Primary**: Envy 500 background, White text
- **Secondary**: Wafer 500 background, White text
- **Outlined**: Transparent background, Envy 500 border
- **Disabled**: 30% opacity

### States
- **Hover**: Darken by one shade (500 → 600)
- **Active**: Darken by two shades (500 → 700)
- **Focus**: 2px outline in Spindle 500
- **Disabled**: 30% opacity

### Borders & Dividers
- **Light**: Envy 100 (#E5EAE1)
- **Medium**: Envy 200 (#CAD4C6)
- **Strong**: Envy 300 (#A5B69F)

## Color Accessibility Matrix

```
                White   Envy50  Envy100 Envy500 Black
Black (#10100F)  AAA     AAA     AAA     AA      -
Envy 500         AA      AA      AA      -       AA
Wafer 500        AA      AA      AA      -       AA
White            -       -       -       AA      AAA
```

Legend:
- AAA: Contrast ratio ≥ 7:1 (Enhanced)
- AA: Contrast ratio ≥ 4.5:1 (Minimum)
- -: Not recommended
